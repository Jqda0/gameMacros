# Game Macros

AutoHotkey v2 macro scripts for PC games. Each game has its own folder with a README covering setup, bindings, and combo reference.

All scripts use AutoHotkey v2 and are scoped to their respective game window — no other applications are affected.

## Requirements

- Windows PC
- [AutoHotkey v2](https://www.autohotkey.com/) (free, open source)

## Usage

1. Install AutoHotkey v2 from [autohotkey.com](https://www.autohotkey.com/)
2. Navigate to the folder for your game
3. Double-click the `.ahk` file — a tray icon appears
4. Launch the game and play
5. Right-click the tray icon → **Exit** to stop
